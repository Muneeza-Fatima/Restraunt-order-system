

🚀 About This Project

Hi!  I built this Restaurant Order System using HTML, CSS, and JavaScript to practice my front-end development skills.

📝 What I did in HTML

Created a responsive navigation bar.

Designed a hero section with an "Order Now" button.

Added a search bar for food items.

Created food cards with images, names, prices, and "Add to Cart" buttons.

Built a cart sidebar to display selected items.

Added a contact section with a contact form.

Created a simple footer for the website.


🎨 What I did in CSS

Designed a modern and responsive user interface.

Used Flexbox and CSS Grid for the layout.

Added hover effects, transitions, and animations.

Styled the navigation bar, hero section, food cards, buttons, cart sidebar, contact form, and footer.

Made the website mobile-friendly using media queries.


 What I did in JavaScript?

Implemented the Add to Cart functionality.

Created a shopping cart sidebar with open and close features.

Added a search feature to filter food items.

Updated the cart count and total price automatically.

Added remove item functionality.

Implemented the checkout process.

Used Local Storage to save cart data even after refreshing the page.


🛠 Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)

Font Awesome


🎯 What I Learned

Through this project, I improved my HTML, CSS, and JavaScript skills. I learned how to build interactive web applications, manage the DOM, handle user interactions, and create a responsive user interface.
